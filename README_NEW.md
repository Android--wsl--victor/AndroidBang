总榜分三：天榜、名榜、遗碑。

* 天榜：或名门正派，或鬼斧神工。天下之大，缺之若失。
* 名榜：十八般武艺，各家有套路。天下之大，得之你幸。
* 遗碑：天下之大，天下之外，亦有天下。
